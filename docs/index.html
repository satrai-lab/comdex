Getting Started with ComDeX

Welcome to the ComDeX Wiki, an architecture designed for efficient data exchange across diverse IoT ecosystems. This quick-start guide will provide:

  • Detailed information about ComDeX.
  • A step-by-step guide to setting up the ComDeX prototype.
  • Links for further understanding and reading.
  • Contact information for any queries or contributions.

Dependencies

Our ComDeX prototype depends on various technologies and tools including:

  • Mosquitto
  • NGSI-LD information model
  • A suitable server or local machine
  • A recent python distribution

Get ComDeX!

Discover the most updated version of ComDeX on our Prototype Implementation page. Our ComDeX UI based usage tool can be found on our ComDeX Usage Tool page.
